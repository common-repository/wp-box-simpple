== Description ==

This plugin Permit to create , edit and delete a simple box .
A box have just a title and content .

To view the box in your blog , you have two possibilities , the first is to add a widget and select your box and the second is to utilize the shortcode call box .
for example [box id=1]

This plugin is a good base of create a news plugin .
